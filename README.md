# 🏠 House Price Prediction
A Machine Learning project to predict house prices based on various features like area, bedrooms, location, and age using Linear Regression.
## 📊 Project Overview

This project uses **Linear Regression** to predict house prices with **97% accuracy**. It includes comprehensive Exploratory Data Analysis (EDA), feature engineering, and interactive visualizations.

**Key Highlights:**
- 🎯 97% R² Score achieved
- 📈 Comprehensive data analysis with multiple visualizations
- 🔍 Feature importance analysis
- 📊 Model evaluation with multiple metrics
- 🐍 Built with Python and Scikit-learn

## 🎯 Objectives

- Predict house prices based on multiple features
- Perform comprehensive Exploratory Data Analysis (EDA)
- Build and evaluate Linear Regression model
- Visualize relationships between features and target variable
- Achieve high prediction accuracy

💡 Key Insights
From the analysis, I discovered:

🏡 Area is the strongest predictor of house price (highest correlation)
📍 Location matters significantly - premium areas command 30-40% higher prices
🆕 Newer houses (< 5 years) have 20% price premium
🛏️ 3-4 bedroom houses are most common and have best resale value
📊 Linear relationship exists between area and price (justifies Linear Regression)
